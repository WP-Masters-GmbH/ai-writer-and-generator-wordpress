== Description ==

**Easy and Free AI Content Assistant for WordPress**

You can easily set a list of topics, click "Sent to Assistant" and get ideas for your creativity.

= What's inside of Free Version available here? =
* AI Generations by given topics
* 3 AI Generations Per Day
* One-Click Import

= What's inside of PRO Version? =
* Unlimited content generation
* Auto Import (Auto Posting)
* [Pro Version download and pricing can be found at official website](https://wp-masters.com/products/ai-content-writer-generator-pro/ "https://wp-masters.com/products/ai-content-writer-generator-pro/")

= Popular use cases =
* You need to write articles but don't know where to get ideas and start from
* You need to quickly get a lot of content

= How to use it =
* Install and Activate AI Writer & Generator plugin
* Use FREE or paid licence code
* Put a list of topics to get ideas for, and related categories
* Click "Send to Assistant" and wait for ideas to be created
* Where content ideas are ready - click "View Idea" or "Import"
