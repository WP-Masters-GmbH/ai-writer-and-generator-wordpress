== Description ==

**Most popular AI Content Assistant for WordPress**

You can easily set a list of topics, click "Sent to Assistant" and get ideas for your creativity.

= What's inside? =
* 20 ideas included in free version
* Content ideas generation by given topics
* Handwork still needed - you need to manually write your posts, but you still save a lot of time with our AI Assistant ideas
* 20+ languages supported

= Popular use cases =
* You need to write articles but don't know where to start from

= How to use it =
* Install and Activate AI Writer & Generator plugin
* Use free or paid licence code
* Put a list of topics to get ideas for
* Click "Send to Assistant" and wait for ideas to be created
* Where content ideas are ready - click "View Idea" and you will be redirected to ai.wp-masters.com to view what AI suggest to start from
